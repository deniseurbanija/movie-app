About this project

This is a movie web site developed using technologies such as Node.js, Express, MongoDB and Tailwind CSS . The web site displays movies through cards, allowing users to add and remove movies from a REST API. Thanks to AJAX, the application offers a real-time interactive experience, while Node.js and Express handle server requests and logic. MongoDB and Mongoose are responsible for storing and manipulating data efficiently, while Tailwind CSS ensures an appealing design across different devices.
